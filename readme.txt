Простая нейросеть на чистом Javascript с использованием библиотеки brain.js с опцией примитивного обучения.
Весь код (разметка, стили и скрипты) находятся в одном .html-файле.
На канвасе можно рисовать фигуры, которые программа, после предварительного обучения, распознает, как Positive или Negative.
Рисование осуществляется ЛКМ.
По нажатию клавиши "C" экран очищается (шрифт английский).
По нажатию клавиши "V" происходит обучение.
По нажатию клавиши "B" - распознавание фигуры программой.
Приятного просмотра!
